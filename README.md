# YUV420 Overlay Tool (C++)

Консольное приложение для наложения BMP-изображения (24-bit RGB) на YUV420-видео.

Проект реализует:
- чтение BMP (24-bit)
- конвертацию RGB → YUV420 в многопоточном режиме
- наложение изображения на каждый кадр
- многопоточную обработку

---

## Поддерживаемые форматы

### Входное видео
- формат: **raw YUV420**
- без заголовков
- передаётся ширина и высота

### Входное изображение
- формат: **BMP**
- глубина: **24-bit (BGR)**

### Выход
- формат: **raw YUV420**

### Требования

```bash
mkdir build
cd build
cmake ..
cmake --build .
```
---
## Запуск

```bash
./yuv_overlay <input.yuv> <width> <height> <overlay.bmp> <output.yuv> <x> <y>
```
### Здесь
- `<input.yuv>` - видео в формате `.yuv`, на которое необходимо наложить изображение
- `<width>` - ширина `<input.yuv>`
- `<height>` - высота `<input.yuv>`
- `<overlay.bmp>` - изображение, которая будет накладываться на видео
- `<output.yuv>` - видео после обработки (после добавления изображения)
- `<x> <y>` - координаты наложения 
---

## Ограничения
- ширина и высота должны быть чётными
- координаты наложения `(x, y)` должны быть чётными
- накладываемая катинка должна полностью помещаться в кадр

---

## Основные модули
- `bmp_reader` - читает BMP-файл и возвращает `RgbImage`
- `rgb_to_yuv` - конвертирует `RGB → YUV420` в многопоточном режиме
- `overlay` - накладывает `YUV420`-изобраение на кадр
- `yuv420` - создание и хранение Y, U, V плоскостей
